.. _intro-install:

==================
Installation guide
==================

Supported Python versions
=========================

ayv requires Python 3.7+, either the CPython implementation (default) or
the PyPy implementation.

.. _intro-install-ayv:

Install ayv
===========

You can install ayv from GitHub or from PyPi

Install from GitHub
-------------------

To install from GitHub, run:

.. code-block::

    pip install ayv
